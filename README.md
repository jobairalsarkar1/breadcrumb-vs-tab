A **dynamic dashboard layout** built with **Next.js**, **TypeScript**, and **Tailwind CSS** featuring a **collapsible sidebar**, **tabbed topbar navigation**, and **breadcrumb-style active tabs** similar to Chrome or VS Code tabs.

## Features

- Collapsible sidebar with icons for Overview, Users, Products, Posts, Blogs, Comments  
- Clicked links open as tabs; multiple tabs can be active  
- Active tab merges with page content (`bg-gray-200`)  
- Rounded bottom corners on neighboring tabs for smooth design  
- Horizontal scroll for overflowing tabs  


## UI / UX Highlights
- Smooth sidebar expand/collapse  
- Tabs behave like modern browser tabs  
- Active tab visually merges with page content  
- Scrollable tabs when exceeding viewport width  
- Close icon for inactive tabs
